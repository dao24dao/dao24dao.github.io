用过 DeepSeek 的人都知道，用 DeepSeek 生成的HTML代码，在电脑上是可以直接运行的（不需要额外操作）

![](image1.png)

但是手机上却无法运行。

这就导致如果手边没有电脑，在手机上写代码调试效果是没法实现的，很不方便。

针对此，我 DeepSeek 写了一个小工具，5 分钟开发完成，可以直接在手机上运行HTML代码，使用超级简单，直接浏览器打开即可。

体验地址： https://dao24dao.github.io/p/html_runner/

当然，你也可以把这个小工具部署在自己的服务器上，随时随地可以访问！！

## 开发步骤
### 1、 输入提示词
```
帮我写一个 html 的页面，要完整代码。
功能：
1. 一个不限长度的输入框，输入框中可以输入 html 代码（包含 css和 js 代码）
2. 输入框下面有两个按钮，分别是“运行”和“清空”;
点击运行，则以浏览器的形式运行输入框中的代码，展示效果；
点击清空，则把输入框中的内容清空。 
3. 整体风格参考 linear app 的风格来做，每个元素都要 有样式。

```

生成之后，可直接点击“在线运行”，看生成的效果。

![](image.png)

**/ 如果有问题，则继续问 DeepSeek，让它帮你解决！**

比如： 点击运行按钮没有反应。 那你就输入：
```
点击“运行按钮”没反应，帮我重新生成。 要完整的代码。
```

DeepSeek 会帮你生成完整的最新代码，继续...
